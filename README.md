# To-Do Client
To-Do Client is a simple and efficient task management backend application built using Python, FastAPI, and Uvicorn.
The project provides RESTful APIs for managing to-do tasks and is designed to be fast, scalable, and easy to extend.

FastAPI ensures high performance and automatic interactive API documentation, while Uvicorn serves as the ASGI server for handling asynchronous requests efficiently.
